# DevStore

Uma aplicação de e-commerce desenvolvida com **Next.js**, utilizando o **App Router** para criar uma experiência moderna, responsiva e otimizada. O projeto simula uma loja virtual com catálogo de produtos, pesquisa, carrinho de compras e páginas de detalhes dos produtos.

## ✨ Funcionalidades

- Vitrine de produtos em destaque
- Catálogo completo de produtos
- Página de detalhes do produto
- Pesquisa de produtos
- Carrinho de compras
- Interface responsiva
- Componentes reutilizáveis
- Navegação otimizada com App Router

## 🚀 Tecnologias

- Next.js
- React
- TypeScript
- Tailwind CSS
- Shadcn UI
- Lucide React
- ESLint
- Cypress

## ⚙️ Como executar

### Clone o repositório

```bash
git clone https://github.com/klayver-dev/devstore.git
```

### Acesse a pasta do projeto

```bash
cd devstore
```

### Instale as dependências

```bash
npm install
```

### Execute a aplicação

```bash
npm run dev
```

A aplicação estará disponível em:

```text
http://localhost:3000
```

## 📦 Scripts

```bash
npm run dev      # Inicia o servidor de desenvolvimento
npm run build    # Gera a build de produção
npm run start    # Executa a aplicação em produção
```
